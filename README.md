# OpenTreeMap Modeling and Prioritization Service

## Getting Started

### Run the service

1. ``git clone git@github.com:OpenTreeMap/OpenTreeMap-Modeling.git``
1. ``cd OpenTreeMap-Modeling``
1. ``./sbt run``

### Build a distribution tarball

1. ``git clone git@github.com:OpenTreeMap/OpenTreeMap-Modeling.git``
1. ``cd OpenTreeMap-Modeling``
1. ``./make-tar``
